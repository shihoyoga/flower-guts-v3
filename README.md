# Flower Guts Project v3 - 腸内環境改善アプリ

腸内環境の改善を目指すための、包括的な診断・記録・管理アプリケーションです。毎日の便の状態、食事、体調、気分などを記録し、腸内環境の改善をサポートします。

## 🌸 特徴

- **複数の診断モード**: 快便チェック、食事記録、体調管理など複数のモードから選択可能
- **詳細な記録機能**: 便の状態、気分、睡眠、水分摂取、食事内容などを詳細に記録
- **わかりやすいUI**: シンプルで使いやすいインターフェース
- **レスポンシブデザイン**: スマートフォンでも快適に利用可能
- **データ共有**: 記録結果をテキストで共有可能

## 🚀 使い方

1. [GitHub Pages](https://shihoyoga.github.io/flower-guts-v3/) でアプリを開く
2. 目的に応じてモードを選択
3. 質問に答えていく
4. 結果を確認・共有

## 📋 記録できる項目

### 快便チェック
- 便の状態（硬さ、量、満足度、臭い）
- グリーンモンスター（野菜摂取）
- 気分
- 睡眠時間
- 水分摂取
- 食事内容
- 体調・肌の状態
- 運動習慣

### その他
- 自由記入欄で詳細なメモが可能
- 日付と名前の記録
